# Todolist hằng ngày

Ứng dụng web mobile-first theo tỷ lệ 9:16 để theo dõi lịch trình trong ngày, tiến độ công việc và tiến độ thời gian hiện tại.

## Chạy ứng dụng

Mở file `index.html` trực tiếp trong trình duyệt.

## Chức năng

- Đồng hồ hiện tại cập nhật theo thời gian thực.
- Highlight công việc đang diễn ra dựa trên giờ hiện tại.
- Checkbox hoàn thành và tự tính tiến độ.
- Thêm, sửa, xóa công việc.
- Sắp xếp danh sách theo thời gian bắt đầu.
- Lưu dữ liệu bằng `localStorage`.
